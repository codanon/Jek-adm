## Tech Stack

Jekyll Admin uses the following technology stack for the front end.

* [React.js](https://facebook.github.io/react/)
* [Redux](http://redux.js.org/)
* [Webpack](https://github.com/webpack/webpack)
* [SASS](https://github.com/sass/sass)
